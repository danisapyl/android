# DanisaOS #

### Sync ###

```bash

# Initialize local repository
repo init -u https://github.com/danisapyl/android -b oreo

# Sync
repo sync
```

### Build ###

```bash

# Set up environment
$ . build/envsetup.sh

# Choose a target
$ lunch aosp_$device-userdebug

# Build the code
$ m
```
